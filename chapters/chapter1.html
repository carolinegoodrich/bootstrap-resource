CHAPTER ONE:

Buckle Up, it's a HTML Crash Course

HTML stands for Hypertext Markup Language. This is the language used to communicate with web browsers what exactly we’re trying to spit out on the screen. To get started, I’m going to go over a few important concepts.

In order to create a web page, you will first open up your favorite text editor (I use Brackets, Live Preview is my best friend but NotePad works just as well) and create a page called index.html.

The .html extension is important because we’re telling your computer/web browser what kind of file it is which dictates how the file is interpreted. Index essentially means, “this is the home page, this is where it’s poppin’ off, it’s lit fam come thru.”

All of the code you write for the browser to interpret needs to be between two tags. This tells the browser where to start reading and then where to stop.

Pull up your favorite website, right click on the page, and then select, “inspect element.” Notice the HTML code. Although I’m sure it’s way more complex than anything we’re going to cover, you will be able to see the main concepts being utilized even on these more detailed websites.

We use “tags” to specify what kind of content we’re trying to show. Based on these tag keywords, the content is represented in a different way. Some basic tags to remember are:

<html>
    <body>
        <h1>Hello world</h1>
       <p> This is a paragraph </p>
    </div>

There have been recent improvements from HTML5 which makes tag keywords more powerful and coding easier.
