CHAPTER TWO:

STYLIN’ AND WILDIN’: An Introduction to CSS

Now that you have a basic understanding of how to get content on a web page, let’s focus on how to make it not look so boring!

In this chapter we’re going to focus on how to organize our code into sections, known as either classes or ID’s, and how to style those sections along with some of the other tags we learned about in the first chapter.

In order to do this, we’re going to start on a little project. I know what you’re thinking, “Oh wow, homework.. cool, Caroline.” Trust me, I know. I’m lame, but we’re going to create this project and build on it throughout this course. That way you literally see the concepts building on one another and you don’t feel cheated and lied to unlike in your math courses.

Create a new folder on your computer, and name it “ABOUTME”. Create another folder inside of that folder, and name that one, “images”. Open that folder up using a text editor of your preference, but if your text editor doesn’t have that option just make sure to save your HTML files inside the ABOUTME file.

Next step: Create an INDEX.HTML file like we did before! We’re going to actually put meaningful content on this page, though.

One way to organize your code into sections is by putting them into containers, so to speak. We do this by using div tags. These tags can either be a div ID tag or a div CLASS tag.

Differences between DIV ID and DIV CLASS:

CLASS

Class can be applied to multiple div tags. An example of this would be if you wanted to style buttons on different pages the same way. You could have a tag that was, 


    < div class=”styledButton”>
. You could then use this to style multiple selections. Class tags can be selected using a “.” in front of the name. Remember this for later!

ID

Div ID can only be applied to ONE section. This can be used if you want to have a specific style that is only applied to one section, for example a sidebar on the homepage that does not show up on any other pages. These can be selected using a “#” in front of the name, like this: 


    < div id=”styledButton”>

Add this to your list of things to remember for later!

Now that you have your code, let’s make it not look so boring!

Create a new file in your about me folder, name it “style.css”. It is important that the file extension ends with .css, this is how we tell the web page that this is a “Cascading Style Sheet” file, and it needs to be loaded in order to show the webpage in a certain way.

Now, we’re going to go back to your index.html file, and in the HEADER portion of your file, before the TITLE section, we’re going to add: 


    <link  href="/style.css" rel="stylesheet">

This tells your index.html page to load the file “style.css” when the page first loads, that way index.html loads looking the way we want it to.
